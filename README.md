# Calculative-Foundation
#  Linear Algebra Project: Student Performance Analysis

##  Project Overview
This project demonstrates the application of linear algebra concepts on a dataset of student scores.  
The dataset is synthetically generated using Python and represents marks obtained by students in different subjects.

The main goal is to apply mathematical concepts such as vectors, matrices, eigenvalues, and dimensionality reduction techniques.

---

##  Dataset
- The dataset contains student scores in subjects:
  - Math
  - Physics
  - Chemistry
  - English
  - Computer
- Total marks are also calculated.
- Dataset is generated using Python (NumPy & Pandas).
